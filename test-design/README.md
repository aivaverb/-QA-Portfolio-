### 🧪 Техники тест-дизайна

В этом разделе собраны примеры применения основных техник тест-дизайна в учебных проектах.

**Классы эквивалентности и граничные значения**  
Техника, которая позволяет разбить все возможные значения на группы (классы) и проверить границы этих групп. Применялась для тестирования поля ввода возраста.

📋 [Подробнее с таблицей](test-design/klassy_ekvivalentnosti.md)

---

**Таблица решений**  
Техника, которая помогает проверить все комбинации условий и соответствующие результаты. Применялась для тестирования формы регистрации.

📋 [Подробнее с таблицей](test-design/tablica_resheniy.md)

---

**Попарное тестирование**  
Техника, которая сокращает количество тестов при большом количестве параметров. Применялась для тестирования формы редактирования пользователя (8 полей → 39 тестов вместо 65 536).

📋 [Подробнее с таблицей](test-design/poparnoe_testirovanie.md)

---

**Состояния и переходы**  
Техника, которая проверяет все возможные состояния объекта и переходы между ними. Применялась для тестирования жизненного цикла товара в интернет-магазине.
