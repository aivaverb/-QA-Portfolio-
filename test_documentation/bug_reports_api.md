## Баг: Удаление чужого комментария → 200 OK (должен быть 403)

**ID:** #18977  
**Статус:** Новая  
**Приоритет:** Немедленный  
**Серьезность:** Критическая  
**Автор:** Елена Вербова  

### Описание
DELETE запрос на чужой комментарий возвращает 200 OK вместо 403 Forbidden.

### Шаги воспроизведения
1. DELETE: `https://dummyapi.io/data/v1/comment/68b348206a9a33266751e1e1`
2. Header: `app-id: 6672cdc851fccc417aad28f5`
3. Нажать "Send"

### Результаты
| Ожидаемо          | Фактически        |
|-------------------|-------------------|
| 403 Forbidden     | 200 OK            |
| Данные сохраняются| Данные удаляются  |

### Воздействие
- Критическая уязвимость безопасности
- Можно удалять чужие комментарии
- Нарушение целостности пользовательского контента

### Рекомендации
- Добавить проверку прав доступа к комментариям
- Верифицировать владельца комментария перед удалением
- Возвращать 403 Forbidden при попытке удалить чужой контент
